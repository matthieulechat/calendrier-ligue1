# BLK-004 — Téléchargement logo Le Mans FC échoue silencieusement (Commons Special:FilePath)

| Friction                                                                                                                                                                                                                                                                   | Cause réelle                                                                                                                                                                                                                                    | Solution                                                                                                                                                 | Statut |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Plusieurs tentatives de récupérer le logo officiel Le Mans FC via `commons.wikimedia.org/wiki/Special:FilePath/...` (différents encodages testés : espaces, underscores, apostrophes) retournaient une page HTML au lieu du SVG attendu, sans message d'erreur exploitable | Le fichier n'existe pas sur Wikimedia Commons — les logos de clubs sportifs (marques déposées) sont hébergés localement sur le wiki de langue (fr.wikipedia.org), pas sur Commons qui est réservé aux médias librement partageables entre wikis | Bascule vers l'API MediaWiki (`action=query&prop=imageinfo&iiprop=url`) sur fr.wikipedia.org pour obtenir l'URL `upload.wikimedia.org` réelle du fichier | résolu |
