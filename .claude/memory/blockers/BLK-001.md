# BLK-001 — Licences des logos clubs (repo luukhopman) : vérifiées, aucune autorisation trouvée

| Friction                                                                                                                                                                                            | Cause réelle                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                | Solution                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           | Statut |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Les logos des clubs Ligue 1 utilisés dans `public/logos/ligue1/` proviennent du repo tiers luukhopman, dont les conditions d'usage n'avaient pas encore été vérifiées (cf. Phase 1.4/7 du ROADMAP). | Vérification effectuée le 2026-07-02 : (1) le repo `luukhopman/football-logos` n'a **aucune licence** déclarée (`license: null` via l'API GitHub, pas de fichier `LICENSE`) — droits d'auteur par défaut, aucun droit de réutilisation accordé explicitement ; (2) les logos eux-mêmes sont des **marques déposées** des clubs — le droit français ne connaît pas de "fair use" à l'américaine ; (3) les CGU de `ligue1.com` interdisent explicitement toute reproduction des marques LFP (Ligue 1, Ligue 2, My Ligue, Coupe de la Ligue) sans autorisation écrite préalable, **sans exception pour un usage personnel ou non commercial**. | Aucune autorisation formelle trouvée nulle part. Le projet doit rester strictement un outil perso/privé (usage individuel, non diffusé) tant qu'aucune démarche explicite n'a été faite auprès de la LFP/des clubs. Avant toute Phase 7 (mise en public), il faudra soit obtenir une autorisation écrite, soit remplacer les crests officiels par une représentation maison (couleurs/initiales du club, sans le logo protégé), soit accepter le risque en connaissance de cause — **ceci n'est pas un avis juridique**, consulter un professionnel si la mise en public est réellement envisagée. | ouvert |
