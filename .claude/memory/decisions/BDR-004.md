# BDR-004 — API-Football invalidée à l'usage, pivot vers football-data.org (Phase 6)

| Décision                                                                                                                                                                                                                                                                                                          | Pourquoi                                                                                                                                                                    | Alternatives considérées                                                                                                                                                                                       | Statut    |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| Spike initial (2026-07-02) : **API-Football** retenue (endpoint `/fixtures`, league ID **61**, plan gratuit) plutôt que football-data.org ou du scraping maison                                                                                                                                                   | Plan gratuit annoncé sans restriction sur la Ligue 1, endpoint unique `/fixtures` retournant fixtures **et** scores, quota 100 req/jour suffisant pour un cron hebdomadaire | football-data.org (également GO au spike, scores différés sur plan gratuit) ; openligadb (**NO-GO** — couverture Ligue 1 abandonnée depuis 2017/2018) ; scraping maison (écarté tant qu'une API viable existe) | remplacée |
| **Pivot (2026-07-18)** : API-Football invalidée à l'usage — le premier appel authentifié a révélé que le plan gratuit **refuse l'accès à la saison en cours**, restriction absente de la doc publique lue au spike. Bascule vers **football-data.org** (code `FL1`, endpoint `/matches`, plan gratuit 10 req/min) | Seule alternative déjà validée GO en Phase 2 ; `scripts/sync-matches.ts` construit et validé directement dessus                                                             | Retenter API-Football sur un plan payant (écarté pour un outil perso) ; scraping maison (toujours écarté)                                                                                                      | actif     |
