# BDR-004 — API-Football retenue comme source de données (Phase 6)

| Décision                                                                                                                                                                                                                          | Pourquoi                                                                                                                                                                                                                                                                                                                                                     | Alternatives considérées                                                                                                                                                                                                                                                                                                                                                          | Statut |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Pour le pipeline de données réelles (Phase 6), utiliser **API-Football** (api-football.com / api-sports.io) — endpoint `/fixtures`, league ID **61** (Ligue 1), plan gratuit — plutôt que football-data.org ou du scraping maison | Plan gratuit couvre la Ligue 1 sans restriction, endpoint unique `/fixtures` retourne fixtures **et** scores, mise à jour live (~15s) même en gratuit, quota 100 req/jour largement suffisant pour un cron hebdomadaire.
