# BDR-011 — Fix du débordement A4 via espacements non-textuels plutôt que taille de police

| Décision                                                                                                                                                                                                                                                                 | Pourquoi                                                                                                                                                                                                                             | Alternatives considérées                                                                                                                                                                                                                                                             | Statut |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------ |
| Pour corriger le débordement de 301,48mm à 297mm, resserrer uniquement les espacements non-textuels (`padding` vertical de `PosterSheet` 8mm→6mm, marges de `PosterLegend`/`MonthBlock`/`PosterFooter`) plutôt que la taille de police ou la hauteur des lignes de match | Ces dernières venaient d'être augmentées à la demande explicite de Baptiste pour la lisibilité (BDR-008) — les retoucher aurait défait ce travail. Les marges/paddings n'ont pas d'impact sur la lisibilité du contenu. | Réduire `py-[1.5mm]` de `MatchRow` ou les tailles de police (écarté : défait la lisibilité récemment obtenue) ; scaling CSS dynamique (`zoom`/`transform`) du poster entier (écarté : complexité inutile alors qu'un ajustement statique suffit, cohérent avec BDR-008) | actif  |
