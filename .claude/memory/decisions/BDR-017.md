# BDR-017 — Lisibilité logo header/footer : flag `Club.logoNeedsBackdrop`, pas de traitement universel ni calcul de contraste

| Décision                                                                                                                                                                                                                                                                                                                                                          | Pourquoi                                                                                                                                                                                                                                                                                                                                 | Alternatives considérées                                                                                                                                                                                                                                                                                                         | Statut |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Ajout d'un flag manuel optionnel `Club.logoNeedsBackdrop?: boolean` dans le modèle de données (`types/match.ts`, `clubs.json`), lu conditionnellement dans `PosterHeader.tsx` et `PosterFooter.tsx` pour appliquer une pastille blanche (`rounded-full bg-white/[92%]`) derrière le logo du club sélectionné. Seule l'entrée `marseille` porte le flag à ce jour. | Un seul club (OM, logo mono-teinte bleu) se fond visuellement dans son propre fond extrait par `colorthief` (cf. [BDR-006](../decisions/BDR-006.md)) — les 17 autres clubs n'ont pas ce problème. Un flag explicite par club est le plus petit changement qui règle le cas réel sans toucher au rendu des clubs qui n'en ont pas besoin. | Pastille blanche appliquée universellement à tous les logos (rejetée par Baptiste après un premier essai — change le rendu de tous les clubs sans qu'aucun autre n'en ait besoin) ; calcul automatique de contraste logo/fond dans `extract-club-colors.ts` (écarté, complexité disproportionnée tant qu'un seul cas est connu). | actif  |
