# BDR-20260704192350-2 — Espacement statique ajusté empiriquement plutôt que distribution flex élastique

| Décision                                                                                                                                                                                                                                 | Pourquoi                                                                                                                                                                                                                                                                                               | Alternatives considérées                                                                                                                                   | Statut |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Pour remplir la page A4 fixe (297mm) du poster, utiliser des valeurs de padding/police fixes ajustées empiriquement (mesurées via `agent-browser`) plutôt qu'une distribution flex élastique (`justify-between`) entre les blocs de mois | Le nombre de matchs par saison Ligue 1 est fixe (34 journées) — le contenu ne varie pas d'une génération à l'autre, une distribution élastique n'apporte donc aucune robustesse réelle et a produit des espacements incohérents entre mois dès que les colonnes ont un nombre de mois/matchs différent | `flex flex-col justify-between` sur les colonnes de mois (essayé en premier, retiré après retour de Baptiste : grands espaces disproportionnés entre mois) | actif  |
