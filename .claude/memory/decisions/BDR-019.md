# BDR-019 — ClubSwitcher `<1024px` : masquage CSS + valeurs forcées via hook `useMediaQuery`, pas de simple masquage visuel

| Décision                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   | Pourquoi                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     | Alternatives considérées                                                                                                                                                                                                                                                                                                                                                                   | Statut |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------ |
| Sous 1024px (Tailwind `lg`), masquer via CSS (`hidden lg:flex`/`hidden lg:inline-flex`) le bouton "Vue tous les clubs" et le switch "Imprimer vide/avec scores" dans `ClubSwitcher.tsx`, et forcer les valeurs effectivement utilisées par le reste de l'app (`effectiveViewMode="single"`, `effectivePrintBlank=false`) via un nouveau hook `useMediaQuery` (`matchMedia` natif, dérivé par l'initializer de `useState` + souscription `change` en effet, même pattern que la règle react-doctor déjà suivie ailleurs dans le projet). L'état brut (`viewMode`/`printBlank`) reste en state React, jamais écrasé — seule la valeur _effective_ passée aux composants de rendu est forcée. | Le menu centré en pilule (`flex-wrap`) wrappait sur 2 lignes à largeur réduite, rendu visuellement cassé (signalé par Baptiste avec capture d'écran). Masquer les contrôles en CSS seul aurait laissé l'utilisateur bloqué avec un état choisi précédemment (vue grille, impression vide) sans aucun moyen d'y revenir tant que les boutons restent cachés — combiner masquage visuel et forçage de la valeur effective règle les deux problèmes en une fois, et l'état brut préservé permet de retrouver le choix initial dès que la fenêtre est réélargie. | Seuil `sm`/`md` Tailwind standard (rejeté — la mesure empirique en headless donnait un seuil de wrap net à 766-768px, mais le rendu réel signalé par Baptiste wrappait déjà à ~910px, cf. [LRN-029](../learnings/LRN-029.md)) ; masquage CSS seul sans forcer l'état sous-jacent (rejeté — laisse l'utilisateur bloqué sur un choix fait à largeur large, inaccessible à largeur réduite). | actif  |
