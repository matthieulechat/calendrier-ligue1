# BDR-003 — Logos officiels via Wikipédia plutôt que ré-extraction V1

| Décision                                                                                                                                                                                                                                                          | Pourquoi                                                                                                                       | Alternatives considérées                                                                                                                                                                                                   | Statut |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| Pour les logos manquants ou de basse qualité (Le Mans FC, Ligue 1 McDonald's), récupérer l'asset vectoriel officiel sur Wikipédia (wiki de langue, ex. fr.wikipedia.org, via l'infobox du club) plutôt que réutiliser/améliorer l'extraction base64 du fichier V1 | Le V1 contenait des logos basse résolution (qualité "capture d'écran"), insuffisante pour une affiche imprimable pixel-perfect | Ré-upscaler l'asset existant (écarté, qualité plafonnée par la source déjà basse-résolution) ; repo luukhopman/football-logos (ne couvre pas Le Mans FC, absent de Ligue 1 depuis longtemps, ni le logo de ligue lui-même) | actif  |
