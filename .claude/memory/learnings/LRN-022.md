# LRN-022 — `min-height` CSS clampe `scrollHeight`, mesure DOM trompeuse

| Pattern observé                                                                                                                                                                                                                                                                                                                              | Contexte                                                                                                                                                                                                                                                                              | Application future                                                                                                                                                                                                                                                            |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `getBoundingClientRect().height` et `scrollHeight` sur un élément avec `min-height` CSS restent plafonnés à ce minimum dès que le contenu réel repasse sous le seuil — deux états de contenu très différents peuvent renvoyer exactement la même mesure, rendant impossible de savoir "de combien ça dépasse encore" une fois sous la barre. | Diagnostic du débordement d'impression A4 de `PosterSheet` (`min-h-[297mm]`) via `agent-browser eval` : plusieurs itérations de resserrement d'espacement ont montré des mesures identiques malgré des changements CSS réels, menant à de fausses conclusions ("ça n'a aucun effet"). | Avant de mesurer la taille réelle d'un contenu sous un `min-height`/`min-width`, désactiver temporairement la contrainte (`el.style.minHeight='0'`) via `eval`, mesurer, puis restaurer la valeur d'origine — sinon la mesure ne reflète que le plancher CSS, pas le contenu. |
