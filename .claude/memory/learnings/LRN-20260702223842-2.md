# LRN-20260702223842-2 — openligadb : couverture ligue affichée ≠ données disponibles, vérifier par appel réel

| Pattern observé                                                                                                                                                                                                                                                                                                         | Contexte                                                                                                                                                                                                                                                | Application future                                                                                                                                                                                                             |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `getavailableleagues` liste bien un raccourci de ligue (`lg1` pour Ligue 1) dans son catalogue, mais un appel réel à `getmatchdata/lg1/2024` retourne un tableau vide — la présence dans la liste des ligues ne garantit pas de données pour la saison courante, seulement qu'un contributeur a un jour ajouté la ligue | Spike de faisabilité Phase 2 (calendrier-ligue1) : évaluation d'openligadb comme source pour le calendrier Ligue 1 26/27 — la ligue existait dans le catalogue mais seule la saison 2017/2018 avait des données réelles, rendant la source inutilisable | Pour toute API de données sportives communautaire/participative, toujours confirmer la couverture par un appel direct sur la saison ciblée (pas seulement lire la liste des ligues déclarées) avant de la retenir comme source |
