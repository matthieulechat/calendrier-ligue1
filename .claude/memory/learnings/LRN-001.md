# LRN-001 — Escalade impression : CSS natif → html2canvas+jsPDF → react-pdf

| Pattern observé                                                                                                                                                                                                                  | Contexte                                                                                                                                                                                                                                                                                                                                                                   | Application future                                                                                                                                                                                                |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Pour une fonctionnalité avec plusieurs approches de complexité croissante (ici l'export/impression d'affiche), tester d'abord la solution native la plus simple en conditions réelles avant d'investir dans une lib plus lourde. | `docs/ROADMAP.md` (Phase 1.3) prévoit un test empirique réel du CSS print (`@page`, `print-color-adjust: exact`) sur plusieurs navigateurs/imprimantes avant tout fallback. `html2canvas`+`jsPDF` réutilise le DOM existant si le CSS print échoue ; `react-pdf` implique une réimplémentation complète du design dans un moteur séparé — réservé en tout dernier recours. | Réutiliser cette hiérarchie (natif → lib légère réutilisant l'existant → lib lourde réimplémentant tout) pour toute fonctionnalité similaire du projet où plusieurs solutions de coût croissant sont disponibles. |
