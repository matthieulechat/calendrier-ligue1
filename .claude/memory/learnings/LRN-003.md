# LRN-003 — Logos/crests de clubs souvent hébergés localement par wiki, pas sur Commons

| Pattern observé                                                                                                                                                                                                                                                                                                                            | Contexte                                                                                                                                                                                                                                                                                       | Application future                                                                                                                                                                                                                                                                                           |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Les logos de clubs sportifs (marques déposées, usage "fair use") sont fréquemment hébergés localement sur le wiki de langue (ex. fr.wikipedia.org) plutôt que sur Wikimedia Commons (partagé entre wikis) — `Special:FilePath` sur Commons renvoie alors une page HTML (souvent 404) au lieu du fichier, sans message d'erreur exploitable | Recherche du logo officiel Le Mans FC : plusieurs tentatives de téléchargement direct via Commons ont échoué silencieusement (fichier HTML au lieu de SVG, différents encodages testés) avant de comprendre que le fichier vivait sur fr.wikipedia.org (cf. [BLK-004](../blockers/BLK-004.md)) | Pour récupérer un asset Wikipédia par programmation, utiliser l'API `action=query&titles=Fichier:X&prop=imageinfo&iiprop=url&format=json` sur le wiki de langue concerné (pas seulement Commons) pour obtenir l'URL réelle `upload.wikimedia.org` — plus fiable que deviner l'encodage de `Special:FilePath` |
