# LRN-028 — Lisibilité connue sur une seule instance → flag manuel sur les données, pas de calcul généralisé prématuré

| Pattern observé                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        | Contexte                                                                                    | Application future                                                                                                                                                                                                                                                                                                                                                          |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Face à un problème de rendu (ici contraste logo/fond) connu sur une seule instance (OM), deux options generalisées ont été écartées au profit d'un flag explicite : un traitement universel (pastille blanche sur tous les logos, change le rendu de tout ce qui n'a pas le problème) et un calcul automatique (contraste logo/fond dans le script de génération de couleurs, complexité disproportionnée pour un seul cas connu). Le flag manuel (`Club.logoNeedsBackdrop`) est le plus petit diff qui règle le cas réel sans effet de bord sur les autres instances. | cf. [BDR-017](../decisions/BDR-017.md) — décision issue de ce raisonnement pour le logo OM. | Avant d'implémenter un calcul généralisé (contraste, distance de couleur, heuristique quelconque) pour corriger un rendu, compter combien d'instances sont réellement concernées aujourd'hui. Un seul cas connu → flag explicite sur la donnée concernée. Ne généraliser en calcul automatique que si un 2e cas apparaît et qu'un pattern commun se dessine entre les deux. |
