# LRN-017 — Une validation empirique de mise en page print n'est valide que pour le contenu testé

| Pattern observé                                                                                                                                                                                                                                                                                                                                                                                                 | Contexte                                                                                                                                                                                                                                                                                                | Application future                                                                                                                                                                                                                                                               |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Une mesure empirique (`getBoundingClientRect` via `agent-browser`) qui confirme qu'un contenu tient dans une page A4 ne garantit ce résultat QUE pour le jeu de données précis mesuré au moment du test. Fusionner ensuite une fonctionnalité qui élargit la variabilité du contenu (ex. sélection d'un club parmi 18 au lieu d'un seul) rend cette garantie invalide sans le signaler dans aucun diff de code. | Le poster calendrier n'avait été mesuré et calé à 297mm que pour Le Mans FC (seul club existant sur la branche `phase-1`). Le merge de la navigation multi-clubs (venue d'une autre branche) a réintroduit un débordement de ~4,5mm, jamais détecté car aucun fichier de code n'avait changé de valeur. | Après avoir fusionné une fonctionnalité qui élargit la variabilité du contenu d'une mise en page print déjà validée empiriquement, revérifier la mesure sur TOUS les cas de variabilité possibles (boucle automatisée via `agent-browser`), pas seulement sur le cas déjà testé. |
