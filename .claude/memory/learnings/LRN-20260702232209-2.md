# LRN-20260702232209-2 — ligue1.com n'héberge pas de crests couleur haute résolution ; conversion SVG→PNG via canvas navigateur sans dépendance

| Pattern observé                                                                                                                                                                                                                                                                                                                  | Contexte                                                                                                                                                                                                                                                                  | Application future                                                                                                                                                                                          |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Le site officiel `ligue1.com` ne propose que des icônes footer monochromes (`/images/clubs/monochrome/L1/*.webp`, 1 seule couleur) et des crests couleur via un CDN tiers (`image.mpg`/MonPetitGazon) plafonnés à 128×128 — ni l'un ni l'autre n'améliore les PNG déjà en place (139×181, cf. [BLK-001](../blockers/BLK-001.md)) | Demande de récupérer des logos "haute qualité" sur ligue1.com pour les 18 clubs — exploration via `agent-browser` (accueil, `/fr/classement`, `/fr/calendar`) : aucune page club dédiée, tentatives de résolutions plus grandes (256×256, 512×512) sur le CDN tiers → 403 | Ne pas repartir chercher sur ligue1.com pour ce projet ; si une meilleure qualité est un jour nécessaire, viser les sites officiels de chaque club (kit presse) plutôt que le site de la ligue              |
| Convertir un SVG en PNG transparent sans ajouter de dépendance (`sharp`/`resvg-js`) : charger le SVG en `data:` URI dans une page, dessiner l'`<img>` sur un `<canvas>`, exporter en `toDataURL('image/png')`                                                                                                                    | Uniformisation du logo Le Mans FC (seul SVG parmi des PNG) vers `.png` — fait via `agent-browser eval` (Canvas API native du navigateur, déjà ouvert pour la QA visuelle du projet) plutôt qu'installer un paquet Node dédié pour une conversion ponctuelle               | Réutiliser ce pattern pour toute conversion SVG→PNG ponctuelle quand un navigateur est déjà disponible dans la session (agent-browser ou équivalent) — évite d'alourdir `package.json` pour un usage unique |
