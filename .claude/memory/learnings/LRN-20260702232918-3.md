# LRN-20260702232918-3 — Toggle écran/impression sans JS : `print:text-transparent` sur un input contrôlé

| Pattern observé                                                                                                                                                                                                                                                                                                                                                                                                               | Contexte                                                                                                                                                                                                                                    | Application future                                                                                                                                                                                                   |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Pour afficher une valeur éditable à l'écran tout en la masquant visuellement à l'impression (ex. mode "brouillon" à remplir au stylo), appliquer une classe Tailwind conditionnelle `print:text-transparent` (+ `print:placeholder-transparent` si besoin) sur l'élément — l'écran reste toujours fonctionnel/rempli, seule la sortie `@media print` masque le texte, sans détection JS de l'impression ni composant dupliqué | Toggle "Imprimer vide" / "Imprimer avec scores" (Phase 5, `ScoreBox`) : les inputs de score restent identiques et éditables à l'écran quel que soit le mode, seule la classe CSS conditionnelle change ce qui sort réellement sur le papier | Réutilisable pour tout composant nécessitant un affichage différent écran/impression sans dupliquer le rendu ni écouter `window.matchMedia('print')` — préférer une classe `print:*` conditionnelle à une logique JS |
