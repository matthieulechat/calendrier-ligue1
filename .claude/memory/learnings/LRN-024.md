# LRN-024 — `break-inside:avoid` empêche la fragmentation print

| Pattern observé                                                                                                                                                                                                                                                                      | Contexte                                                                                                                                                                                                                                 | Application future                                                                                                                                                                                                                                                                     |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Sur un document destiné à l'impression, un bloc de texte à cheval sur une limite de page peut se scinder visuellement (texte tronqué/dupliqué entre les deux pages) même quand le dépassement total est minime — symptôme distinct et plus visible qu'un simple débordement de page. | `PosterFooter` (texte légal + timestamp) se fragmentait à la limite de page 1/2 lors des tests d'impression réels de Baptiste, produisant un texte visuellement coupé/dupliqué, avant même de corriger le dépassement de hauteur global. | Sur tout bloc qui ne doit jamais être coupé entre deux pages à l'impression (footer, ligne de tableau, carte), ajouter `break-inside: avoid` (Tailwind : `break-inside-avoid`) de façon préventive — force le bloc à basculer entier sur la page suivante plutôt que de se fragmenter. |
